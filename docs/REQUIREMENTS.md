## 1. 产品愿景（待完善）

**一句话：**  



**我最常用的 场景：**

1. 撰写学习笔记，这个笔记我希望可以有预览功能，用类似博客的形式，去预览，然后平时写的时候还是笔记本的形式
一篇笔记中需要记录很多形式的信息，比如
  - 学习资源来源/参考：文档连接、视频网站连接、
  - 结构化的文本，多级标题，这里按照层级写具体的知识点内容，希望可以支持markdown格式
  - 示意图：可以是截图、可以是类似飞书里的结构图（思维导图那种），需要在笔记中随意拖动位置，这个我想的是通过无限画板实现，在画板中可以随意放大缩小拖拽图片
  - 视频嵌入：最好就是直接可以在笔记中播放
  - 便签、任务表等小工具，也可以随意拖拽
  - 字体换色、大小、粗细、格式刷等
  - 最后需要有ai 辅助功能，比如有一个输入栏和ai对话，让ai辅助修改、生成等等，甚至我可以让ai去某些网站学习，然后直接总结出来
2. 平时随手记录，备忘录，可能记录的内容非常杂
  - 突然的灵感
  - 临时的规划，todo项
  - 记录哪一天要干什么，可以及时提醒我
  - 随时记录一些东西
3. 账单整理
  - 我希望能够自动记录每一笔花销、分个类别，总结统计，还可以提醒自己
4. 几个人的私密朋友圈：
  - 几个人在这里记录，互相可以看见每个人最近记录了什么，评论之类的
  - 可以存放照片
5. 个人网站
  - 可以从笔记后台中挑选笔记上传到个人博客网站，我希望这个笔记本中也能直接切换到博客预览

**参考产品（勾选或补充）：**

- Notion（块编辑 + `/` 命令）
- Typora（纯 Markdown 流式）
- Apple 备忘录（极简快记）
- 飞书 / 语雀（富文本 + 结构）
- Obsidian（本地 + 双链）
- OneNote（无限画板，多端数据互通）
- github上非常好看的博客个人网站项目
